## üéØ Usage Examples

### Basic Calculations
```
2 + 3 = 5
10 * 5 = 50
100 / 4 = 25
```

### Scientific Functions
```
sin(30) = 0.5
cos(60) = 0.5
log(100) = 2
sqrt(16) = 4
5! = 120
```

### Complex Expressions
```
(2 + 3) * 4 = 20
sin(45) + cos(45) = 1.414
log(100) * sqrt(16) = 8
```

### Memory Operations
```
1. Enter a number: 100
2. Press M+ to add to memory
3. Enter another number: 50
4. Press M- to subtract from memory
5. Press MR to recall memory value
6. Press MC to clear memory
```
